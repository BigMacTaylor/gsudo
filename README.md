# Gsudo
A simple tool to run graphical apps with sudo. It was built and tested on [sway wm](https://github.com/swaywm/sway).

![Gsudo](https://github.com/BigMacTaylor/gsudo/blob/main/screenshots/gsudo.png "gsudo")

### Example usage
```bash
gsudo pcmanfm
```

WARNING! It is highly recommended not to run graphical apps with sudo.
Doing so can cause serious problems, such as currupt file permissions in your home directory,
which can prevent you from logging in. Please use with caution.
